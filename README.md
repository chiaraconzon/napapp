# NapApp

NapApp is an intelligent application designed to optimize daytime rest for adolescents.  
The app analyzes the user's daily schedule and sleep parameters, including sleep debt, to suggest the most suitable time and optimal duration for a nap.

The goal is to support recovery from sleepiness while avoiding interference with nighttime sleep, adapting recommendations to the user's individual needs.

## Main Features

### Intelligent Nap Scheduling
A dedicated algorithm analyzes the user's daily context and suggests the ideal nap duration and timing based on the actual need for recovery.

### Daily Activity Management
An integrated calendar considers lessons, study sessions, workouts, and other daily activities, identifying the best opportunities for rest.

### Sleep Analysis
The app uses sleep parameters, including sleep debt, to estimate the user's recovery needs and provide personalized nap recommendations.

### Accessibility
The interface is available in both Italian and English.
